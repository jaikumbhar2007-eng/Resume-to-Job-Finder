# Resume-to-Job Finder 🚀

An AI-powered, full-stack web application that automates the job search process by parsing user resumes and matching them with real-time job postings.

## 🌟 Features
* **AI Match Engine:** Utilizes Natural Language Processing (Scikit-Learn/TF-IDF) to analyze resume keywords against live job descriptions.
* **Real-Time Data:** Integrates with SerpApi to fetch live Google Jobs data dynamically.
* **Modern UI:** Clean, responsive glassmorphism frontend built with HTML/CSS.
* **Production Ready:** Fully containerized and deployed on Render.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-Learn (TF-IDF Vectorizer), NumPy, Pandas
* **APIs:** SerpApi (Google Jobs API)
* **Frontend:** HTML5, CSS3
* **Deployment:** Render, Gunicorn
